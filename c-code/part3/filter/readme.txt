Command: ./filter -flag input.bmp output.bmp

FLAGS
    -g: Grayscale
    -r: Reflect horizontally
    -f: Reflect vertically
    -e: Edge detection
    -b: Blur image
    -s: Apply sepia filter
    -i: Invert image
    -c: Cipher image
    -d: Decipher image

Note that the code only works for .BMP images.
